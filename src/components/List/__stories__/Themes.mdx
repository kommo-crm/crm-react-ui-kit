## Theming with Text component

`List` inherits theme tokens from its parent `Text` component. Pass `theme` to `Text` — the `List` inside picks it up automatically without needing its own `theme` prop.

```jsx
import { Text } from '@kommo-crm/crm-react-ui-kit/Text';
import { List } from '@kommo-crm/crm-react-ui-kit/List';
import { TextPrimaryTheme } from '@kommo-crm/crm-react-ui-kit/Text';

function App() {
  return (
    <Text size="l" theme={TextPrimaryTheme}>
      <List type="bulleted">
        <List.Item>List item 1</List.Item>
        <List.Item>List item 2</List.Item>
      </List>
    </Text>
  );
}
```
